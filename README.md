# 🛒 Shopzyy Billing System

A simple supermarket billing system built in Python while learning **Harvard's CS50: Introduction to Computer Science**.

This project is being improved after every lecture to apply the concepts I learn. Instead of creating many small projects, I am continuously developing this one into a complete billing system.

---

## 📚 Concepts Used

- Functions
- `main()` function
- Variables
- User Input
- Conditionals (`if`, `elif`, `else`)
- Loops (`while`)
- Lists
- Dictionaries
- Returning values from functions
- Input validation

---

## ✨ Current Features

- Welcome screen
- Customer name input
- Product menu display
- Item selection using item number
- Quantity selection
- Price calculation
- Item total calculation
- Purchase multiple items
- Running grand total
- Input validation for menu choices

## ▶️ How to Run

1. Clone the repository

```bash
git clone https://github.com/yourusername/shopzyy-billing-system.git
```

2. Navigate to the project

```bash
cd shopzyy-billing-system
```

3. Run the program

```bash
python billing.py
```
